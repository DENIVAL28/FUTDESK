
# 🏟️ FuteDesk - Gestão Esportiva

O **FuteDesk** é um sistema web desenvolvido para simplificar a organização de campeonatos esportivos regionais. Focado na praticidade, ele permite gerenciar tabelas, times, resultados e classificações de forma dinâmica.

## 🚀 Funcionalidades
* **Gestão de Torneios:** Cadastro e organização de campeonatos (Futebol, Futsal, Vôlei, etc.).
* **Tabelas de Jogos:** Geração automática e acompanhamento de resultados.
* **Classificação em Tempo Real:** Cálculo de pontos, vitórias e saldo de gols.
* **Interface Administrativa:** Painel para controle total dos dados.

## 🛠️ Tecnologias Utilizadas
* **Backend:** Python com [Flask](https://flask.palletsprojects.com/)
* **Banco de Dados:** SQLite
* **Frontend:** HTML5, CSS3 e Bootstrap
* **Hospedagem:** PythonAnywhere
